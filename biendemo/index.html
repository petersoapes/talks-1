API overview


  • RESTful API on top of BIEN SQL database

  • Ruby Sinatra framework (same behind our Fishbase API)

  • Goals: a) make machine-machine conversation easy b) which makes maintaing client libraries easier, and c) makes SQL database changes easy to do

features


  • Authentication: API key

  • w/API keys: user level stats on usage, deal with abusive users, etc.

  • Proper HTTP codes, etc: should fail well :)

  • JSON and CSV response formats

  • Caddy server

  • mirrors BIEN R package methods

  • (hopefully) good performance
